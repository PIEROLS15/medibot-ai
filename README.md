## ğŸ› ï¸ Installation

### Prerequisites

Make sure you have [Node](https://nodejs.org/en/) installed on your system:

```bash
node -v
```

### Project Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/PIEROLS15/medibot-ai.git
   cd medibot-ai
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure environment variables:**

   The project includes an example file .env.example. Copy it to create your .env file and adjust the values according to your local environment:
   ```bash
   cp .env.example .env
   ```
   Make sure to modify the necessary values, for example:
   ```bash
   DATABASE_URL="postgresql://user:password@localhost:5432/recommendation_system_db?schema=public"
   NEXTAUTH_SECRET="your_secret_key"
   NEXTAUTH_URL="http://localhost:3000"
   ```
   âš ï¸ **Important: Replace user, password, and the database name with your actual credentials**
   
4. **Create the database in PostgreSQL:**

   From your SQL client or terminal:
   ```bash
   CREATE DATABASE recommendation_system_db;
   ```
   
5. **Generate the Prisma client:**
   ```bash
   npx prisma generate
   ```
   
6. **Run migrations to create tables:**
   ```bash
   npx prisma migrate dev
   ```
   
7. **Run seed data:**
   ```bash
   npm run seed
   ```

8. **Start the development server ğŸš€:**
   ```bash
   npm run dev
   ```

   The server will be available at `http://localhost:3000` and will connect to the database using the variables from the `.env` file.

   ```bash
   git clone https://github.com/<your_username>/medibot-ai.git
   cd medibot-ai
   git checkout -b feature/new-api
   ```

2. **Clone your fork and create a new branch for your feature:**
   ```bash
   git fork https://github.com/PIEROLS15/medibot-ai.git
   ```

3. **Add your new module or functionality**

   If it's an API, place it inside:
   ```bash
   src/app/api/
   ```

   Example:
   ```bash
   src/app/api/user/route.ts
   ```
   
4. **Run ESLint before committing (automatically fixes errors):**
   ```bash
   npm run lint:fix
   ```
   
5. **Commit your changes with a descriptive message:**
   ```bash
   git commit -m "feat: add new user API"
   ```

6. **Push your branch to the remote repository:**
   ```bash
   git push origin feature/new-api
   ```

7. **Open a Pull Request (PR)**
   
   - Go to the original repository:
     https://github.com/PIEROLS15/medibot-ai

   - Briefly explain the changes made and their purpose.
